Title: Instant Dictionary Web App
Description: A Web App that let the Users type in a term in the text box
and return the English Definition of the term instantly as soon as the User Finished Typing.
The Web App consist of the Website with the Navigational Menu, A Home, A Dictionary
and the About Page.
